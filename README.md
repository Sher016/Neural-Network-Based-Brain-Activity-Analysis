Este repositorio presenta un enfoque de análisis de actividad cerebral mediante redes neuronales, aplicado a señales EEG previamente procesadas.Se usó un dataset real. El objetivo es diseñar y entrenar modelos capaces de identificar patrones sutiles que no se detectan con métodos estadísticos convencionales, ofreciendo una comprensión más profunda de la complejidad de las señales cerebrales y explorando posibles correlaciones entre grupos de estudio.
